# BioTech Dashboard - Agricultura Sustentável

O BioTech Dashboard é uma aplicação web desenvolvida para auxiliar no monitoramento de recursos agrícolas e na tomada de decisões relacionadas à irrigação inteligente.

O sistema permite ao usuário informar a umidade atual do solo e a previsão do tempo, realizando uma análise automática das condições para recomendar ações de irrigação. O objetivo é promover o uso eficiente da água, reduzir desperdícios e incentivar práticas agrícolas sustentáveis.

## Funcionalidades

* Monitoramento da umidade do solo.
* Análise das condições climáticas.
* Recomendações automáticas de irrigação.
* Exibição de diagnóstico do sistema.
* Estimativa de economia de água.
* Apresentação de práticas sustentáveis para a agricultura.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Contribuir para uma agricultura mais sustentável por meio da tecnologia, auxiliando produtores na gestão eficiente dos recursos hídricos e na preservação do meio ambiente.

## Autor

Projeto desenvolvido para fins educacionais e demonstração de conceitos de monitoramento agrícola e sustentabilidade.
